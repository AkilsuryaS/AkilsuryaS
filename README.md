## About me

I’m an AI/ML builder with a strong interest in turning ideas into working products and reproducible experiments. My projects range from tool-using LLM applications and multilingual NLP to computer vision, scientific imaging, and small models designed to run on everyday hardware.

I enjoy the full journey: understanding the problem, designing the data and model, making inference practical, validating the result, and documenting the work so someone else can learn from it.
